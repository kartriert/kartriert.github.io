Website

Eine wichtige Frage, die man beim Erstellen einer Website klären sollte, ist, ob man eine dynamische oder statische Website erstellen möchte. Dynamische Webseiten werden benutzt, wenn viel mit Nutzenden interagiert wird, wie bei einem Online-Shop zum Beispiel. Da der Fokus unserer Website darauf liegt, Informationen zu präsentieren, und sie keine Daten von Nutzenden sammelt, haben wir uns für eine statische Website entschieden. Dadurch, dass wir Github benutzen, um das Projekt und den gesamten Workflow zu verwalten, war es für uns am sinnvollsten, die Website mit Github Pages zu erstellen und so direkt in unser Github-Projekt einzubinden.


Für das Design und die Optik der Website haben wir uns für ein vorgefertigtes Template entschieden, bei dem wir die verschiedenen Elemente (wie zum Beispiel die Dropdown-Funktion weiter oben), die Anordnung und die Farben durch Bearbeitung der jeweiligen CSS-Dateien angepasst haben. Die Website wird auch über ein Github-Repository verwaltet und ermöglicht so das Einbinden von Daten und, in unserem Fall, das Einbauen einer interaktiven Karte. Sonstiger Inhalt, wie etwa dieser Text, wurde als HTML in die jeweiligen Dateien eingebunden. Das verwendete Hintergrundbild auf der Startseite ist von Jakob Braun (kostenlos zu verwenden im Rahmen der Unsplash Lizenz).

Das Logo für das Projekt hat Lisa Braune mit viel Liebe erstellt.
